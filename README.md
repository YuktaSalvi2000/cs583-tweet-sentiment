# cs583-tweet-sentiment

CS583 research project: sentiment classification of Obama and Romney tweets into positive (1), negative (-1), and neutral (0) classes.

## Installation & Setup

```bash
# Clone repository
git clone https://github.com/YuktaSalvi2000/cs583-tweet-sentiment.git
cd cs583-tweet-sentiment

# Create virtual environment (Windows)
python -m venv venv
.\venv\Scripts\Activate.ps1

# Create virtual environment (macOS / Linux)
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```

Note: Initial development and experimentation were performed using Google Colab.  
The requirements.txt file mirrors the Colab environment to support local reproducibility.
